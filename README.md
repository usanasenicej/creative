# Creatix Agency - Premium Landing Page

A pixel-perfect, modern landing page built with Vue 3 (Vite), featuring a dark theme with vibrant neon green accents.

## ✨ Features

- **Responsive Design:** Fully optimized for desktop, tablet, and mobile.
- **Glassmorphism:** Modern UI elements with blur and transparency.
- **Micro-animations:** Smooth transitions, hover effects, and scroll-triggered reveals.
- **High-Quality Visuals:** Uses curated Unsplash photography to ensure a premium look.
- **Modular Components:** Clean, reusable Vue components.

## 🚀 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## 🛠 Tech Stack

- **Framework:** Vue.js 3 (Composition API)
- **Tooling:** Vite
- **Styling:** Vanilla CSS (Variables, Flexbox, Grid)
- **Typography:** Outfit (Google Fonts)
